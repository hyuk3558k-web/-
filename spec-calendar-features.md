# Spec: 달력 기능 개선 및 신규 기능

## Overview
월별 달력의 일정 관리 UX를 개선하고, 이미지 다운로드 접근성 확대, 칭찬스티커 관리자 권한 강화를 구현한다.

## Goals
- 달력에서 일정이 있는 날짜를 빨간점으로 표시
- 일정 세부정보를 모달이 아닌 달력 아래에 인라인으로 표시
- 관리자(PIN 인증)만 일정 등록/수정/삭제 가능
- 지난 일정 클릭 시 해당일 입력값 조회 가능
- 가족 구성원이 업로드한 이미지를 누구나 클릭하여 다운로드 가능
- 칭찬스티커 등록/삭제는 관리자만 가능 (현재 이미 구현됨)

## Requirements
### Functional
1. **달력 빨간점 표시**: 이벤트가 있는 날짜에 빨간점(`#F47458`) 표시 (기존 파란점 `#6B9FD6` -> 빨간점으로 변경)
2. **달력 하단 세부정보**: 날짜 클릭 시 EventModal(팝업) 대신 달력 아래 영역에 해당일 이벤트 목록 표시
3. **관리자 전용 일정 관리**: 일정 추가/수정/삭제 버튼은 관리자 PIN 인증 후에만 노출
4. **지난 일정 조회**: 과거 날짜 클릭 시에도 해당일 입력값(이벤트) 확인 가능
5. **이미지 다운로드 개방**: HomeworkChecklist에서 `isAdmin` 조건 없이 photoUrl이 있으면 누구나 다운로드 버튼 표시
6. **칭찬스티커 관리자 제한**: 이미 AdminPinModal로 구현됨 - 변경 불필요

### Non-functional
- 기존 Firebase Firestore 연동 유지
- 모바일 반응형 레이아웃 유지

## Technical Notes
- **Calendar.jsx**: 이벤트 dot 색상을 `#6B9FD6` -> `#F47458`(빨간점)으로 변경
- **Dashboard.jsx**: EventModal 제거, 달력 아래에 인라인 이벤트 상세 영역 추가. 관리자 PIN 인증 상태 관리 추가
- **EventModal.jsx**: 더 이상 팝업이 아닌 인라인 컴포넌트로 리팩토링 -> 새 컴포넌트 `EventDetail.jsx`로 대체
- **HomeworkChecklist.jsx**: 다운로드 버튼의 `isAdmin && item.photoUrl` 조건에서 `isAdmin` 제거
- **useEvents.js**: 변경 없음

## UI/UX Notes
- 달력 날짜 클릭 -> 달력 아래에 해당일 이벤트 목록 슬라이드 표시
- 관리자 미인증 시: 이벤트 조회만 가능 (읽기 전용)
- 관리자 인증 시: 이벤트 추가/수정/삭제 폼 노출
- 빨간점은 달성률 dot과 나란히 표시 (기존 레이아웃 유지)

## Tradeoffs & Decisions
- EventModal(팝업) 방식을 완전히 제거하고 인라인 방식으로 전환 (사용자 요청)
- 이미지 다운로드를 전체 공개로 변경 (가족 공유 목적)
