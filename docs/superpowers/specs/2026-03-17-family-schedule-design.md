# 서씨네 시간표 - 설계 문서

## 개요

어린이 3명(서주원, 서예원, 서채원)의 일과를 효율적으로 관리하는 가족용 웹앱.
가족 누구나 접속해서 시간표 확인, 숙제 완료 체크, 달성률 확인이 가능하다.

## 기술 스택

- **Frontend**: React 18 + Vite + Tailwind CSS
- **Backend/DB**: Firebase (Firestore + Storage + Hosting)
- **푸시 알림**: Firebase Cloud Messaging (FCM)
- **텔레그램 알림**: Telegram Bot API
- **폰트**: NanumSquareRound (Google Fonts)
- **배포**: Firebase Hosting (무료 플랜)

## 화면 구성

### 1. 메인 대시보드 (`/`)

- **오늘 날짜** 표시
- **세 아이의 달성률 카드**: 작은 바(progress bar) 형태로 오늘 숙제 달성률 표시
  - 각 카드는 아이별 테마 컬러 적용
- **칭찬 스티커 현황**: 각 아이가 모은 스티커 수 표시 (별, 하트, 왕관 아이콘)
  > ⚠️ Updated: 스티커 발급 권한이 관리자 전용으로 변경됨
  - **관리자만 스티커 발급 가능**: 스티커 부여 버튼 클릭 시 4자리 PIN 입력 팝업으로 관리자 인증 필요
  - **관리자 스티커 제거 가능**: 관리자 인증 후 기존 스티커를 삭제할 수 있음 (스티커 항목에 삭제 버튼 표시)
- **월간 달력**: 날짜별 셀에 달성률 색상 표시
  - 90%+ 초록, 50~89% 노랑, 50% 미만 빨강
  - 날짜 클릭 시 해당 날의 상세 달성 내역 표시
  > ⚠️ Updated: 달력에 가족 공통 일정 등록 기능 추가
  - **일정 등록 기능**: 달력의 날짜를 클릭하여 가족 공통 일정을 추가할 수 있음 (누구나 등록 가능, 별도 인증 불요)
    - 일정 항목: 제목, 시간(선택), 메모(선택)
    - 등록된 일정은 달력 셀에 점(dot) 또는 라벨로 표시
    - 일정 수정/삭제도 가능

### 2. 개인 시간표 탭 (`/schedule/:name`)

상단 탭으로 주원/예원/채원 전환.

- **요일별 시간표** (월~토): 타임라인 형태로 표시
  - 학교 (회색)
  - 학원 (아이별 테마 컬러)
  - 숙제 시간 (연한 노랑)
  - 자유 시간 (연한 초록)
  - 저녁 식사 (오렌지)
- **숙제 체크리스트**: 각 숙제 항목에 완료 체크박스
  - 체크하면 달성률에 자동 반영
  - 본인이 직접 체크
  - 숙제 완료 시 **사진 인증 업로드** 가능 (선택사항, 카메라/갤러리에서 선택)
  - 업로드된 사진은 숙제 항목 옆에 썸네일로 표시

#### 숙제 자동 배치 규칙

- **학원 전날 숙제**: 다음 날 학원이 있는 경우, 전날 시간표에 해당 학원 숙제 1시간 자동 삽입
- **숙제 우선순위 체계**:
  - **필수 숙제**: 학원 전날 해당 과목 숙제 (반드시 시간표에 포함)
  - **선택 숙제**: 국어+한자, 러닝포털 등 (시간 여유 있을 때 배치, 없으면 주말로 이동)
- **복수 숙제 배치**: 필수 숙제를 먼저 배치하고, 남는 시간에 선택 숙제 배치
  - 서주원: [필수] 수학 1시간(학원 전날) + 엠베스트 1시간(매일)
  - 서예원: [필수] 영어 단어+워크북 30분(영어학원 전날) + 수학 1시간(수학학원 전날) / [선택] 러닝포털 30분 + 국어+한자 30분
  - 서채원: [필수] 영어 단어+워크북 30분(영어학원 전날) + 수학 1시간(수학학원 전날) / [선택] 러닝포털 30분 + 국어+한자 30분
- 빈 시간대 중 가장 이른 시간에 배치. 빈 시간이 부족하면 선택 숙제는 주말(토)로 자동 이동

#### 숙제 데이터 생성

- 사용자가 해당 날짜 페이지에 처음 접속할 때 자동 생성
- 요일별 고정 숙제 템플릿 기반으로 생성
- 이미 생성된 날짜는 재생성하지 않음

#### 달성률 계산

- **공식**: (완료된 숙제 항목 수 / 전체 숙제 항목 수) × 100%
- 숙제가 없는 날(일요일 등)은 달성률 표시하지 않음
- 자정이 지나면 미체크 항목은 미완료로 확정
- 달성률은 **필수 숙제 기준**으로 계산 (선택 숙제 완료 시 보너스 표시)

#### 푸시 알림

- Firebase Cloud Messaging(FCM) 활용, 브라우저 알림 허용 시 동작
- **50% 미만 알림**: 저녁 7시 기준으로 필수 숙제 달성률이 50% 미만이면 해당 아이 기기에 푸시 알림 발송
  - 알림 내용: "오늘 숙제가 아직 많이 남았어요! 화이팅!"
- 관리자가 알림 시간 조정 가능

#### 숙제 사진 인증

- 숙제 완료 체크 시 카메라/갤러리에서 사진 업로드 가능 (선택사항)
- Firebase Storage에 저장 (`/photos/{childId}/{date}/{homeworkId}.jpg`)
- 숙제 체크리스트에서 사진 썸네일 확인 가능
- 관리자/가족이 사진 확인 가능
> ⚠️ Updated: 관리자 사진 다운로드 기능 추가
- **관리자 사진 다운로드**: 관리자 모드에서 숙제 사진을 개별 클릭하여 원본 다운로드 가능
  - 사진 썸네일 옆에 다운로드 아이콘 버튼 표시 (관리자 인증 상태에서만 노출)
  - 다운로드 파일명: `{아이이름}_{날짜}_{숙제명}.jpg` 형식

### 3. 관리자 모드 (`/admin`)

- 간단한 비밀번호 입력으로 진입 (4자리 PIN)
- **시간표 편집**: 고정 시간 추가/수정/삭제
  - 학교, 학원, 반복 일정 관리
- **숙제 항목 편집**: 숙제 종류와 소요 시간 설정
- **스티커 관리**: 스티커 발급 및 제거 (관리자 전용)
  > ⚠️ Updated: 스티커 발급/제거가 관리자 전용으로 변경됨

## 데이터 구조 (Firestore)

```
/children/{childId}
  - name: string ("서주원")
  - color: string ("#6B9FD6")
  - grade: string ("중1")

/schedules/{childId}/weekly/{dayOfWeek}
  - blocks: [
      { start: "09:00", end: "15:30", type: "school", label: "학교" },
      { start: "15:30", end: "19:00", type: "academy", label: "수학학원" },
      { start: "17:00", end: "17:50", type: "lesson", label: "피아노" },
      { start: "19:00", end: "20:00", type: "lesson", label: "책수업" },
      ...
    ]

/homework/{childId}/daily/{date}
  - items: [
      { id: string, label: "수학 숙제", duration: 60, completed: boolean },
      { id: string, label: "엠베스트", duration: 60, completed: boolean, required: true, photoUrl: string|null },
      ...
    ]

/stickers/{childId}
  - total: number
  - history: [
      { from: string, type: "star", date: timestamp },
      ...
    ]

/events/{eventId}
  - title: string ("가족 여행")
  - date: string ("2026-03-20")
  - time: string|null ("14:00")
  - memo: string|null
  - createdAt: timestamp

/settings
  - adminPin: string
  - dinnerTime: "19:30"
  - telegramBotToken: string
  - telegramChatId: string (관리자 개인 채팅 ID)
  - telegramSendTime: "19:00"
```

## 디자인 시스템

### 컬러 팔레트

- **배경**: 아이보리 `#FDF8F0`
- **카드 배경**: 화이트 `#FFFFFF`
- **주원 테마**: 소프트 블루 `#6B9FD6` / 연한 `#E8F1FA`
- **예원 테마**: 소프트 핑크 `#E8859A` / 연한 `#FDE8ED`
- **채원 테마**: 소프트 민트 `#6BC5B8` / 연한 `#E5F5F2`
- **포인트 액센트**: 웜 코랄 `#F47458`
- **텍스트**: 다크 브라운 `#3D3229`
- **서브 텍스트**: `#8C7B6B`

### 일정 블록 컬러

- 학교: `#D4D0CC` (회색)
- 학원: 아이별 테마 컬러
- 숙제: `#FFF3CD` (연한 노랑)
- 자유시간: `#D4EDDA` (연한 초록)
- 저녁식사: `#FFE0CC` (연한 오렌지)
- 책수업/피아노: `#E8D5F5` (연한 보라)

### 스타일 가이드

- 카드: `border-radius: 16px`, `box-shadow: 0 2px 12px rgba(0,0,0,0.06)`
- 버튼: `border-radius: 12px`, 따뜻한 코랄 포인트
- 여백: 섹션 간 `48px`, 카드 내부 `24px`
- 폰트: NanumSquareRound, 제목 Bold, 본문 Regular
- 아이콘: 둥글고 친근한 스타일 (Lucide React)

### 레퍼런스

- Superrbook 랜딩페이지 참고: 따뜻한 톤, 둥근 카드, 넉넉한 여백, 파스텔 배경

## 인증/접근

- 일반 접근: 별도 로그인 없이 누구나 시간표 확인, 숙제 체크 가능
- 관리자 모드: 4자리 PIN 입력 (시간표 수정 시에만)
- Firebase 익명 인증으로 세션 관리

## 반응형 디자인

- **모바일 우선** 설계 (가족 대부분 스마트폰으로 접속)
- 모바일: 카드 1열, 달력 축소형, 시간표 세로 스크롤
- 태블릿/PC: 카드 3열, 달력 풀사이즈, 시간표 가로 배치

## 일정 블록 type 정의

- `school`: 학교
- `academy`: 학원 (수학, 영어 등)
- `lesson`: 개별 레슨 (피아노 17:00-17:50, 책수업 19:00-20:00)
- `homework`: 숙제 시간
- `free`: 자유 시간
- `dinner`: 저녁 식사
- `break`: 쉬는 시간

## 일요일/공휴일 처리

- 일요일: 기본적으로 쉬는 날 (숙제/달성률 없음)
- 공휴일: 관리자가 특정 날짜를 "휴일"로 지정 가능 (학교 블록 제거)

## 텔레그램 달성률 알림 (신규)

- **매일 저녁 7시** 자동으로 관리자 개인 텔레그램 채팅에 달성률 요약 발송
- Telegram Bot API 활용 (봇 토큰 + 채팅 ID를 관리자 설정에서 입력)
- **발송 내용**:
  ```
  📊 오늘의 달성률 요약 (2026-03-19)
  ━━━━━━━━━━━━━━━━
  🔵 서주원: 80% (4/5 완료)
  🩷 서예원: 100% (6/6 완료) ⭐
  🟢 서채원: 50% (3/6 완료) ⚠️
  ━━━━━━━━━━━━━━━━
  전체 평균: 77%
  ```
- 달성률 100%인 아이에게는 ⭐ 표시, 50% 미만은 ⚠️ 표시
- Firebase Cloud Functions 또는 클라이언트 측 스케줄링으로 구현
- 관리자 페이지에서 봇 토큰, 채팅 ID, 발송 시간 설정 가능
